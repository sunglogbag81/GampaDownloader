YouTube Downloader

대기열 기반 다운로드 + 필터링 + 코덱/품질 설정 (개인용 GUI)

Windows 다운로드 GitHub

공지: 지원 중단

이 프로그램은 더 이상 업데이트/지원하지 않습니다. (현재 배포된 버전은 계속 다운로드 가능하지만, 동작을 보장하지 않습니다.)

  • Windows 환경에서 Chrome/Edge 쿠키를 자동으로 가져오는 방식(yt-dlp --cookies-from-browser)이 브라우저 보안/정책 변경으로 인해 정상 동작하지 않는 경우가 있습니다.
  • 대표적으로 Chrome/Edge 쿠키 DB 잠금 문제(#7271) 또는 DPAPI 복호화 실패(#10927)로 인해 “쿠키를 켰는데도 분석/다운로드가 실패”할 수 있습니다.
  • 관련 이슈: Chrome 쿠키 DB 잠금 문제 (#7271), DPAPI 복호화 실패 (#10927)

참고: 쿠키가 없어도 되는 경우

  • 일부 일반(공개) 영상은 쿠키 없이도 다운로드가 가능합니다.
  • 다만 사용자/네트워크/YouTube 정책 변화에 따라 공개 영상도 추가 인증이 필요해질 수 있습니다.

대안 안내

  • Firefox 쿠키로는 정상 동작하는 경우가 많습니다. (yt-dlp 이슈에서도 “Use Firefox”가 Windows의 주요 워크어라운드로 안내됩니다.)
  • Chrome/Edge만 사용하는 환경에서는, 쿠키를 확장프로그램 등으로 내보내서 --cookies 방식으로 사용하는 것이 대안으로 안내됩니다.
  • 또는 4K Video Downloader 같은 전용 프로그램을 이용하는 것도 방법입니다.

주요 기능(마지막 버전 기준)

  • URL 여러 개/채널 탭 큐로 추가
  • Shorts 제외, 키워드 포함/제외, 날짜 필터
  • 품질 선택(최대 해상도 제한), 코덱 선호 설정
  • ffmpeg/ffprobe 동봉(병합/후처리)

사용 방법

  1. exe 실행
  2. 저장 폴더 선택
  3. URL 입력 → 대기열 추가 → 다운로드 시작

주의

  • 이 앱은 yt-dlp/ffmpeg를 사용하며, 사용자는 각 서비스 약관과 저작권을 준수해야 합니다.
  • Windows SmartScreen에서 경고가 뜰 수 있어요(서명되지 않은 exe).
  • 환경(브라우저/Windows/YouTube 정책) 변화로 인해 일부 영상은 다운로드가 실패할 수 있습니다.
