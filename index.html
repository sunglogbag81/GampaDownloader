공지: 지원 중단

이 프로그램은 더 이상 업데이트/지원하지 않습니다. (현재 배포된 버전은 계속 다운로드 가능하지만, 동작을 보장하지 않습니다.)

  • Windows 환경에서 Chrome/Edge 쿠키를 자동으로 가져오는 방식(yt-dlp --cookies-from-browser)이 브라우저 보안 변경으로 인해 정상 동작하지 않는 경우가 있습니다. (예: 쿠키 DB 잠금/복호화 실패)
  • 이 문제는 앱 자체 버그라기보다 브라우저/OS 보안 정책과 연동된 이슈로, 사용자 환경에 따라 재현/해결이 어려워 안정적인 서비스 제공이 불가능하다고 판단했습니다.
  • 관련 이슈: Chrome 쿠키 DB 잠금 문제 (#7271), DPAPI 복호화 실패 (#10927)

대안 안내

  • 가장 쉬운 해결책은 4K Video Downloader 같은 상용/전용 프로그램을 사용하거나, 브라우저/OS/네트워크 환경에 맞는 다른 툴을 사용하는 것입니다.
  • 개발자/고급 사용자는 yt-dlp의 최신 가이드(예: PO Token 관련)를 참고해 환경에 맞게 직접 구성할 수 있습니다.

주요 기능(마지막 버전 기준)

  • URL 여러 개/채널 탭 큐로 추가
  • Shorts 제외, 키워드 포함/제외, 날짜 필터
  • 품질 선택(최대 해상도 제한), 코덱 선호 설정
  • ffmpeg/ffprobe 동봉(병합/후처리)

사용 방법

  1. exe 실행
  2. 저장 폴더 선택
  3. URL 입력 → 대기열 추가 → 다운로드 시작

주의

  • 이 앱은 yt-dlp/ffmpeg를 사용하며, 사용자는 각 서비스 약관과 저작권을 준수해야 합니다.
  • Windows SmartScreen에서 경고가 뜰 수 있어요(서명되지 않은 exe).
  • 환경(브라우저/Windows/YouTube 정책) 변화로 인해 일부 영상은 다운로드가 실패할 수 있습니다.
